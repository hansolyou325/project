##프로젝트 설명 및 구조
어플리케이션과 서버, 서버와 도어락이 각각 Wifi 로 통신하며, 어플리케이션과 도어락은 직접적으로 통신하지 않는다. 
사용자는 서버에서 필요한 정보를 조회하고, 서버에서 인증 과정을 거친 후에 DB 상의 정보를 변경할 수 있다. 
도어락은 서버로부터 도어락 개폐 정보에 대한 정보를 얻어와 상태를 반영한다.
